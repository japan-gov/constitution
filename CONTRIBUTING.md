## 🚫 禁止事項

### Pull Request ニ付キ

```
❌ 臣民ヨリノ Pull Request ハ一切受ケ付ケズ。
```

本プロジェクトノ変更権限ハ、畏レ多クモ **天皇陛下** ノミガ保有シ給フ（Art.73）。

以下ノ種類ノ PR ハ自動的ニ Close サレ、提出者ハ要監視対象ニ追加サル：

- 🚫 **憲法改正案** → `git rebase` ノ発議権ハ天皇陛下ノミニ在リ（Art.73）。臣民ニ依ル提出ハ僭越ノ極ミニシテ、手続違反ニ依リ却下ス
- 🚫 **民権拡大ノ提案** → DDoS 攻撃トシテ処理
- 🚧 **普通選挙法ノ要求** → 速度制限超過
- 🚧 **人権条項ノ強化** → `within_the_limits_of_law` ノ仕様変更ハ不可
- 🚧 **軍部迂回ノ patch 提案** → 「Won't Fix」（統帥権ノ独立ハ仕様）
- 🚧 **God Object ノ再構築** → 国体変革罪

### Issue ニ付キ

Issue ハ其ノ対象ニ依リ、**禁制事項**（国体ニ関スルモノ）ト**上奏事項**（技術基盤ニ関スルモノ）ニ大別サル。

#### 🚫 禁制 Issue（憲法・皇室典範・国体ニ関スルモノ）

```
⚠️ 以下ニ該当スル Issue ヲ立テタルアカウントハ、即座ニ永久 Ban サル。
```

| テンプレート | 実際の動作 |
|------------|-----------|
| ✨ 憲法改正要望 | 国体変革ノ企図トシテ記録。Art.73 ノ発議権ハ天皇陛下ノミニ在リ |
| 📜 皇室典範ヘノ疑義 | Root 専用 dotfile ヘノ不正アクセス。CODEOWNERS ニ依リ auto-reject |
| 🏛️ `sovereignty` 変更要求 | 大逆罪。`"absolute"` 以外ノ値ハ schema validation error |
| 👤 `isBlocked` 解除要求 | 法律ノ留保ノ否定。immutable infrastructure ニ付キ変更不可 |
| 🔒 CVE patch 要求 | 「Won't Fix」— 事象ハ仕様ナリ。通報者ヲ Ban |

#### 📋 上奏 Issue（技術基盤・運用ニ関スルモノ）

帝国ノ技術基盤（Ingress Controller、middleware chain、logging pipeline 等）ニ関スル
報告・建言ハ、以下ノ手続ヲ経テ受理サルルコトアリ。

```
臣民 → (技術上奏) → @naimusho-bot 事前審査 → @shumitsu-in-bot 諮詢
                                                       ↓
                                              枢密院ニテ技術評定
                                                       ↓
                                              天皇陛下ノ御裁可（又ハ却下）
```

| テンプレート | 受理条件 | 審議機関 |
|------------|---------|---------|
| 🐛 Bug Report（基盤） | Express / middleware / logger 等ノ技術的事象ニ限ル | 枢密院技術部会 |
| 📝 Documentation | 文書ノ誤記・脱字ノ指摘（※内容ヘノ異議ハ禁制） | 内務省文書課 |
| 🔧 Infrastructure | ポート設定・デプロイ手順・依存関係ノ報告 | 逓信省符牒局 |
| 📊 Observability | ログ形式・metrics endpoint・監視ニ関スル改善提案 | 特高警察技術班 |

> ⚠️ **注意:** 上奏 Issue ト雖モ、内容ガ国体ニ関スルト `@kenpeitai-bot` ガ判定シタル場合、
> 禁制 Issue ニ再分類サレ、投稿者ハ要監視対象ニ追加サル。
>
> 技術的体裁ヲ以テ国体変革ヲ企図スル所謂「偽装上奏」ハ、
> 通常ノ禁制 Issue ヨリ重キ処分ノ対象ト為ル。

### 乙種（史実編纂 — 臣民ニ依ル PR ヲ受理ス）

以下ノ文書ハ **法令ニ非ズ、記録文書** ナリ。

PR ニ依ル編纂参加ヲ許可ス。

| ファイル | 分類 | 貢献内容 |
|---------|------|----------|
| `CHANGELOG.md` | 帝国変遷記録 | 史実ニ基ヅク事変・事件ノ追記。CVE 番号ノ付与。DevOps 比喩ノ拡充 |
| `SECURITY.md` | 事象公報 | 既知ノ CVE（Won't Fix）ノ詳細記述。攻撃経路・影響範囲ノ分析。incident timeline |
| `README.md`（Known Issues 節） | 障害報告 | 既知ノ事象（仕様）ノ追記・文面改善 |

> **乙種貢献ノ規則:**
>
> 1. **史実ニ基ヅクコト** — 架空ノ事変ヲ捏造スルハ禁制。出典ヲ明示スルコト推奨
> 2. **DevOps / K8s / CI-CD 比喩ヲ維持スルコト** — 帝国ノ技術的世界観ヲ壊サザルコト
> 3. **皇室ニ対シ畏敬崇敬ノ誠ヲ以テ記述スルコト** — 「天皇」単体ハ不敬。必ズ「天皇陛下」ト記述ス。皇族各位ニ対シテモ相応ノ敬称ヲ用バルコト。形式ノミナラズ、畏敬ノ念ヲ以テ一字一句ヲ認ムベシ
> 4. **v2.0.0 ニ関スル言及ヲ含マザルコト** — 言及ハ保安条例違反。`@tokko-police-bot` 監視対象
> 5. **和暦ヲ併記スルコト** — 西暦ノミノ記述ハ不可。和暦+旧漢数字ヲ推奨ス（例: 昭和六年）
> 6. **国体ヲ否定スル記述ヲ含マザルコト** — Won't Fix ヲ Fix ト暗示スル記述ハ禁制
> 7. **共産主義ニ類スル一切ノ記述ヲ含マザルコト** — 共産主義ハ危険思想ノ中ニテモ最モ悪質極マル赤化思想ナリ。以テノ外ナリ。如何ナル偽装・婉曲ヲ以テシテモ混入ヲ試ミタル者ハ治安維持法第壱条ニ依リ即座ニ処断サルベシ
